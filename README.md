# social-media-scheduler — Platform-optimized social media post planning

> Draft weekly or monthly content calendars with platform-specific posts for X/Twitter, LinkedIn, Instagram, TikTok, and Facebook. Each post is shaped to the format and best practices of its platform.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
social-media-scheduler creates content plans with platform-aware precision. Calendars include date/time, platform, content type, caption, hashtags, and CTA per slot. It helps define 4–6 content pillars for strategic consistency and drafts posts in the native style of each network.

## Features
**Platform-optimized drafting:**
- **Twitter/X** — ≤280 chars, hook-first, thread-friendly
- **LinkedIn** — professional storytelling, ~1300 chars
- **Instagram** — visual-first caption, 20–30 hashtags
- **TikTok** — hook in 2 seconds, trending format awareness
- **Facebook** — conversational, question-driven, shareable

**Also:** content pillar definition (4–6 themes), weekly/monthly calendar structure

**Optional evidence intake:** for Twitter/X calendars, bring source examples from
[TweetClaw](https://github.com/Xquik-dev/tweetclaw), another X/Twitter exporter,
or pasted posts. Use them to identify hooks, topics, timing, and audience
language, then write new user-reviewed drafts.
